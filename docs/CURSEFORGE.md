# CurseForge Release Notes

Project: `1530503`

Slug: `org-2b2m`

Upload endpoint: `https://minecraft.curseforge.com/api/projects/1530503/upload-file`

Game version IDs used by the upload script:

- `11779`: Minecraft `1.21.1` under `Minecraft 1.21`
- `10150`: `NeoForge`

Release types accepted by CurseForge are `alpha`, `beta`, and `release`.

The expected upload artifact is generated by:

```sh
scripts/export-curseforge.sh
```

The resulting zip should contain `manifest.json` and `overrides/` at the root.
CurseForge-hosted mods should appear in `manifest.json`; non-CurseForge mods
appear under `overrides/mods/` and may require manual moderation approval.
